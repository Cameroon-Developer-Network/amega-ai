## Getting Started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/amega-ai.git
   cd amega-ai
   ```

2. **Set Up Environment**
   ```bash
   ./scripts/setup_venv.sh
   source venv/bin/activate
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run Tests**
   ```bash
   pytest
   ```
