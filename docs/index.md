## üöÄ Quick Start

```bash
# Clone the repository
git clone https://github.com/amega-ai/amega-ai.git

# Set up environment
cd amega-ai
./scripts/setup_venv.sh

# Install dependencies
pip install -r requirements.txt

# Run the application
python -m src.amega_ai.main
```
